# gostarterapp

prereq

```bash
go get -u github.com/spf13/cobra
```
build the application

```bash
go build -o gostarterapp main.go
```

Run the Application:

```bash
./gostarterapp --env dev --project OD --token your_bearer_token
```



```bash
./gostarterapp -e test -p ODS-IDL -t token123
```
